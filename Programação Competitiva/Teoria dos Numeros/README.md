# Divisores
### Divisores de um número
- Como $`n=\sqrt n \sqrt n`$, cada divisor menor que $`\sqrt n`$ tem seu correspondente maior que $`\sqrt n`$;
- Complexidade: $`O(\sqrt n)`$
